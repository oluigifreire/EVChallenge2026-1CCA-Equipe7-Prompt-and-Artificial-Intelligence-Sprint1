# EVChallenge2026-1CCA-Equipe7-Prompt-and-Artificial-Intelligence-Sprint1
Nome do projeto: EV DashBot
- Integrantes: Luigi Freire RM569958 ; Victor Vidigal RM571318; Gabriel Savoy RM568991; Raphael Tien RM570261.
- Problema abordado: Falta de tratamento dos dados extraídos dos EV Chargers para otmização.
- proposta do chatbot: Gerar Dashboard para facilitar visualização dos dados, analisar dados para prever comportamentos futuros e otimizar gargalos atuais, e automatizar otimizações das estações de carregamento.
Tecnologias selecionadas e justificativa técnica: 
- ChatGPT: Trazer Inteligência e memória para o projeto, e apresentar os dados em linguagem natural.
- Plotly: Gerar Dashboards para melhor visualização dos dados.
